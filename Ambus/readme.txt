Things to keep in mind before using:
	1. Only have one of any given cape you want to augment in inventory while augmenting.
		e.g. Don't have two Andartia's Mantles in your inventory while using. only the one you want augmented
	2. Don't use the command to augment if you have already maxed out current augment path. If you use the command with maxxed 		path, it will waste an augment item if you have it available

Commands:
	//ambus item <Thread/Dust/Sap/Dye> <Cape Name> <Path Name>
		e.g. //ambus item Sap Rosmerta's Cape FC
		This sets the augment to use
		Refer to tables before for correct Path Name for given item.
		
	//ambus go
		e.g. augments the cape given selected item, cape, and path.
